# rst-visualizations
Visualizations of sequences selected by RST for compression from select SPEC2017 benchmark memory dumps. The dumps were collected using gcore and randomly sampled using rejection sampling from non-zero pages. You can directly download these files and open them using your browser. We are currently revising the software implementation that supports the tool that generated the current visualization. We will include it in a future update. We sincerely apologize for the inconvenience and thank you for your patience.
